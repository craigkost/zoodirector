Node Tree (Context Menu Options)

The node tree context menu can be accessed by right clicking a node of interest. Certain options below are accompanied by hot-keys (or keyboard shortcuts, if you prefer). Please see below for more details on the context menu options.

Modification

Add
Create the specified zookeeper path including all nodes that do not exist. Will create the node relative to the selected node. Specifying a path beginning with / will create the node using an absolute path. It will create all parent segments which do not presently exist in zookeeper.If the node already exists it will open in the edit panel. Quick access via the Insert key.
Delete
Deletes the selected node including all descendant nodes. Confirmation is requested. Can also be trigger by the Delete key. Using Ctrl + Delete will delete the selected node and all descendants without confirmation. This operation cannot be undone, use at your own risk!
Prune
Deletes the selected node including all descendant nodes as well as all ancestor nodes which only have a single child. Confirmation is requested. This operation cannot be undone, use at your own risk!
Trim
Deletes all descendant nodes. Confirmation is requested. This operation cannot be undone, use at your own risk!

Viewing

Expand All
Expand current node and all descendant nodes. Can also be trigger by * on the selected node.
Collapse All
Collapse current node and all descendant nodes. Can also be trigger by / on the selected node.

Watches

Add Watch
Add a watch for the selected node. Can also be trigger by w on the selected node or Ctrl + w to add watches for the selected node and all descendant nodes.
Remove Watch
Remove watch from the selected node. Can also be trigger by r on the selected node or Ctrl + r to remove all watches for the selected node and all descendant nodes.

Node Edit Panel (Panel Options)

The following are the operations that can be performed within the node edit panel. This panel is populated by selecting a node in the node tree or when a node is created. If the node no longer exists in zookeeper the edit panel will prevent editing and an error will be displayed in the log panel.
Reload
Reload the node's data from zookeeper.
Clear
Clear the node's data from the edit panel. This does not update the data in zookeeper.
Save
Save the node's data to zookeeper.
